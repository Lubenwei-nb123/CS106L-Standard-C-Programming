## 汇编语言

| 优点                 | 缺点                                                         |
| -------------------- | ------------------------------------------------------------ |
| 指令集相对简单       | 完成简单任务也需要编写大量代码                               |
| 性能高               | 可读性差                                                     |
| 可以完全控制程序底层 | 和处理器架构紧密相关，无法跨平台<br />（x86上能执行的代码无法迁移到ARM上） |

## C

出于汇编语言的缺点，Ken Thompson和Dennis Ritchie在1972年创造了C，其优缺点如下：

| 优点     | 缺点                     |
| -------- | ------------------------ |
| 简洁     | 没有类和对象的概念       |
| 性能高   | 难以编写泛型代码         |
| 可跨平台 | 编写大型程序时仍较为麻烦 |

## C++

C++由Bjarne Stroustrup在1983年创建，旨在结合C语言的高效性和简单性，并加入面向对象等高级特性。

C++的**设计理念**如下：

- 只在解决实际问题时添加新特性
- 强调模块化
- 允许程序员在需要时完全控制程序
- 除非万不得已，不牺牲性能
- 尽可能在编译时强制执行安全性
- 程序员应有选择自己编程风格自由
